# 📚 Mini Wikipedia RAG Q&A Bot

> A lightweight Retrieval-Augmented Generation (RAG) bot that answers your questions by pulling relevant information from **Wikipedia** and responding using **Hugging Face Transformers**!


---

## 🚀 Features

- 🔍 **Semantic Search** using Sentence Transformers (`all-MiniLM-L6-v2`)
- 🤖 **Question Answering** with `distilbert-base-cased-distilled-squad`
- 🌐 **Live Wikipedia Search** via Python's `wikipedia` library
- 🧠 Automatically picks the **most relevant chunk** for accurate answers
- 💻 Clean and responsive **Streamlit UI**
- ✅ Runs **100% locally** – No API key or internet dependency for model inference

---

## 🧰 Tech Stack

| Tool | Description |
|------|-------------|
| [Streamlit](https://streamlit.io/) | Lightweight Python web framework for UI |
| [Hugging Face Transformers](https://huggingface.co/transformers/) | Pretrained QA model |
| [SentenceTransformers](https://www.sbert.net/) | For semantic embedding and similarity |
| [Wikipedia](https://pypi.org/project/wikipedia/) | To pull summary content for RAG |
